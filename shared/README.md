# 🛠️ Shared Resources & Tools

본 폴더는 하부 프로젝트 간의 코드 중복을 최소화하고, 개발 효율성을 높이기 위한 공용 자원 및 표준 도구를 포함합니다.

---

## 📂 1. 공용 도구 (`/tools`)

### **[upload.py](./tools/upload.py)**
- **역할:** MicroPython 환경에서 펌웨어를 장치로 즉시 전송하는 고성능 업로드 스크립트.
- **특징:** 
  - `mpremote` 의존성 없이 순수 시리얼 통신으로 작동.
  - 장치의 REPL 상태를 자동 감지하여 인터럽트(`Ctrl+C`) 후 원격 쓰기 수행.
  - 전송 완료 후 자동 리셋 기능을 포함하여 개발 사이클 단축.

## 📂 2. 설계 표준 (Planned)
- **데이터 포맷 표준:** 센서 간 데이터 교환을 위한 공용 JSON 스키마 정의 예정.
- **로그 규격:** 시스템 장애 추적을 위한 공통 로그 메시지 포맷팅 모듈.

---
## 💡 활용 가이드
새로운 제어 보드를 추가할 때, `/shared/tools/upload.py`를 활용하여 일관된 배포 프로세스를 유지할 수 있습니다.
