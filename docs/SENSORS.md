# Hunet 센서 연결 가이드

## 공통 하드웨어

- **MCU**: Raspberry Pi RP2040 Zero
- **RS485 모듈**: TTL-to-RS485 자동 방향 제어 모듈 (VCC / TX / RX / GND / A / B)
- **공통 설정**: 9600 baud, Modbus RTU, 8N1
- **버스**: 모든 센서가 같은 A/B 선에 병렬 연결

### RS485 모듈 ↔ RP2040 배선

| RS485 모듈 핀 | RP2040 핀 | 비고 |
|---|---|---|
| VCC | 5V (VBUS) | |
| GND | GND | 공통 GND |
| TX | **GP1** (RX) | 교차 연결 주의 |
| RX | **GP0** (TX) | 교차 연결 주의 |
| A | 센서들 A+ 병렬 | |
| B | 센서들 B- 병렬 | |

## 센서 주소 할당표

| addr | 센서 | 상태 |
|---|---|---|
| 0x01 | 온습도 센서 | ✅ 운영 중 |
| 0x02 | 토양 5 in 1 | ✅ 운영 중 |
| 0x03 | 조도(일사량) 센서 PR-300AL-RA-N01 | ✅ 운영 중 |
| 0x04 | CO2 센서 CD10 (INFWIN) | ✅ 운영 중 |
| 0x05~ | 추가 센서 예정 | - |

## 1. 온습도 센서

- **파일**: `rp2040_temp_humidity.py`
- **프로토콜**: RS485 Modbus RTU
- **Baud Rate**: 9600
- **주소**: 0x01

### 배선

| 센서 선 색 | 연결 |
|---|---|
| 검정 | GND |
| 갈색 | 5V |
| 노랑 | RS485 모듈 A |
| 파랑 | RS485 모듈 B |

### Modbus 레지스터 (데이터)

| 레지스터 | 데이터 | 단위 | 스케일 |
|---|---|---|---|
| 0x0000 | 습도 | % | ÷ 10 |
| 0x0001 | 온도 | °C | ÷ 10 |

### 출력 예시
```
Temp: 26.4C | Humidity: 30.5%
```

## 2. 토양 5 in 1 센서 (ComWinTop / Halisense 계열)

- **파일**: `rp2040_soil.py`
- **프로토콜**: RS485 Modbus RTU
- **Baud Rate**: 9600 (기본 4800에서 변경 완료)
- **주소**: 0x02 (기본 0x01에서 변경 완료)

### 배선

| 센서 선 색 | 연결 |
|---|---|
| 빨강 | 5V |
| 검정 | GND |
| 노랑 | RS485 모듈 A |
| 파랑 | RS485 모듈 B |

### Modbus 레지스터 (데이터, FC=0x03)

| 레지스터 | 데이터 | 단위 | 스케일 |
|---|---|---|---|
| 0x0000 | 토양 수분 | % | ÷ 10 |
| 0x0001 | 온도 | °C | ÷ 10 |
| 0x0002 | EC | μS/cm | - |
| 0x0003 | pH | - | ÷ 10 |
| 0x0004 | 질소 (N) | mg/kg | - |
| 0x0005 | 인 (P) | mg/kg | - |
| 0x0006 | 칼륨 (K) | mg/kg | - |

### Modbus 레지스터 (설정, FC=0x06으로 write)

| 레지스터 | 설정값 | 비고 |
|---|---|---|
| 0x07D0 | Slave ID (1~254) | 주소 변경 |
| 0x07D1 | Baud rate (0=2400 / 1=4800 / **2=9600**) | baud 변경 |

### 설정 변경 명령 예시
```
# baud → 9600
01 06 07 D1 00 02 59 46

# 주소 → 0x02
01 06 07 D0 00 02 08 86

# slave ID 조회 (브로드캐스트)
FF 03 07 D0 00 01 91 59
```

### 출력 예시
```
Moisture:18.5% | Temp:21.9C | EC:93 | pH:5.7 | N:3 P:6 K:15
```

### 주의사항
- EC=0이면 NPK도 0 (전도도 기반 측정)
- 수분 15% 이상, 흙에 완전히 꽂힌 상태에서 정상 수치 측정 가능
- NPK는 간이 측정값으로 참고용

## 3. 조도(일사량) 센서 (PR-300AL-RA-N01)

- **파일**: `rp2040_solar.py`
- **프로토콜**: RS485 Modbus RTU
- **Baud Rate**: 9600 (기본 4800에서 변경 완료)
- **주소**: 0x03 (기본 0x01에서 변경 완료)
- **전원**: DC 7~30V (12V SMPS 사용)
- **측정 범위**: 0~1800 W/m²

### 배선

| 센서 선 색 | 연결 |
|---|---|
| 갈색 | 12V VCC |
| 검정 | GND |
| 녹색 | RS485 모듈 A |
| 파랑 | RS485 모듈 B |

### Modbus 레지스터 (데이터, FC=0x03)

| 레지스터 | 데이터 | 단위 | 스케일 |
|---|---|---|---|
| 0x0000 | 일사량 | W/m² | - |
| 0x0052 | 편차값 | W/m² | 0~1800 |

### Modbus 레지스터 (설정, FC=0x06으로 write)

| 레지스터 | 설정값 | 비고 |
|---|---|---|
| 0x07D0 | Slave ID (1~254) | 주소 변경 |
| 0x07D1 | Baud rate (0=2400 / 1=4800 / **2=9600**) | baud 변경 |

### 출력 예시
```
Solar:450 W/m2
```

## 4. CO2 센서 (INFWIN CD10)

- **파일**: `rp2040_co2.py`
- **프로토콜**: RS485 Modbus RTU
- **Baud Rate**: 9600
- **주소**: 0x04 (기본 0x02에서 변경 완료)
- **전원**: DC 12~30V
- **측정 범위**: 0~5000 ppm (모델에 따라 다름)
- **웜업 시간**: 최대 180초

### 배선

| 센서 선 색 | 연결 |
|---|---|
| 빨강 | 12V VCC |
| 검정 | GND |
| 노랑 (T+) | RS485 모듈 A |
| 흰색 (T-) | RS485 모듈 B |
| 초록 | 미연결 (아날로그 출력) |

### Modbus 레지스터 (데이터, FC=0x04)

| 레지스터 | 데이터 | 단위 | 스케일 |
|---|---|---|---|
| 0x0000 | CO2 농도 | ppm | - |

### Modbus 레지스터 (설정, FC=0x06으로 write)

| 레지스터 | 설정값 | 비고 |
|---|---|---|
| 0x0200 | Slave Address (0~255) | 주소 변경, 재전원 필요 |
| 0x0201 | Baud rate (3=9600bps) | baud 변경, 재전원 필요 |

### 출력 예시
```
CO2:700 ppm
```

## 새 센서 추가 방법

1. 새 센서 baud rate를 9600으로 변경 (설정 레지스터 확인)
2. 새 센서 주소를 다음 빈 addr로 변경 (현재 0x03부터 가능)
3. A/B 선에 병렬 연결
4. `rp2040_main.py`에 read 함수 및 파싱 추가
5. `web_monitor.py` PATTERN 정규식 및 카드 UI 추가
6. `docs/SENSORS.md` 센서 주소 할당표 업데이트
