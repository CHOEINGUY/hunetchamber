🖥 디스플레이 측 빵판 배치 — RP2350-Touch-LCD-4 + W5500

양액 시스템과 별도 회사망에 LAN으로 연결
⚠ 확인 필요: Waveshare 위키 접속이 막혀서 RP2350-Touch-LCD-4의 정확한 외부 GPIO 핀맵을 가져오지 못함. 아래 GP 번호는 일반 RP2350 보드 패턴 기반의 추정치임. 실제 보드의 외부 헤더 핀 배치를 위키 또는 보드 PCB 실크스크린에서 직접 확인 후 수정해야 함.
⚠ LCD가 내부적으로 점유한 GP 번호 피해야 함 (보통 GP0~GP19 영역). 외부 헤더는 GP20 이상 또는 specific 핀이 노출됨.

📌 다음 단계
1. Waveshare 위키 직접 접속 (브라우저로 www.waveshare.com/wiki/RP2350-Touch-LCD-4) → 외부 헤더 핀맵 사진 또는 PDF 다운로드.
2. LCD 점유 핀 파악 → 회피해서 W5500용 SPI 4핀 + CS 1핀 = 총 5핀 확보.
3. 핀 번호 확정 후 이 HTML의 ⚠ 마크 부분 수정해서 영구 참고용으로 보관.
4. 코드 측: W5500 라이브러리 (WIZNET5K 또는 MicroPython 내장) + HTTP 클라이언트 (urequests) 설치.
5. 네트워크 설정: Pico의 IP 알아내서 디스플레이 코드에 하드코딩 또는 DHCP+mDNS.

🔌 회로 핵심
• 디스플레이 측 W5500 → SPI 4선 + 3V3 + GND = 총 6선만 결선
• 디스플레이는 5V USB 또는 외부 어댑터로 전원 공급 (양액 시스템과 별개)
• LAN 케이블 한 가닥으로 회사망 스위치 연결
• Pico W의 W5500과 같은 네트워크에 있으면 통신 가능
