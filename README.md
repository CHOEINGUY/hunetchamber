# Hunet Chamber Project Workspace (2026)

본 워크스페이스는 농장 환경 제어 및 냉장고 정밀 제어 시스템의 통합 관리 저장소입니다. 최인규 연구원이 수행한 하드웨어 설계, 제어 알고리즘 검증 및 데이터 파이프라인 구축 결과가 정리되어 있습니다.

---

## 프로젝트 핵심 성과 요약 (Executive Summary)

### 1. 냉장고 정밀 제어 및 비대칭 밴드 로직 검증
실제 기동 테스트를 통해 냉각 관성(Undershoot)을 분석하고, 목표 온도 수렴을 위한 비대칭 제어 전략을 수립했습니다. 상세한 실험 데이터와 통찰은 아래 보고서에서 확인할 수 있습니다.

**[핵심 문서] [냉장고 연구 결과 보고서 (Research Summary)](./projects/fridge_controller/docs/07_RESEARCH_SUMMARY.md)**

*   **실험 결과:** 15.0°C 목표 설정 시, 비대칭 밴드(+1.1°C / -0.5°C) 적용을 통해 실제 평균 온도 14.9°C 도달 성공.
*   **보호 로직:** 컴프레셔 소손 방지를 위한 300초 최소 ON/OFF 가드 로직 가동.
*   **안전 장치:** 센서 Stale(유실) 감지 시 제어 홀드 및 DB NULL 처리 로직 구현.

### 2. 농장 게이트웨이 및 데이터 파이프라인 안정화
*   **센서 통합:** 온습도, 토양, 조도, CO2 4종 센서의 RS485 Modbus RTU 통합 수집.
*   **자동 복구:** W5500 이더넷 통신 장애 시 하드웨어 자동 재기동(Self-healing) 메커니즘 적용.
*   **고해상도 로깅:** 1Hz(초당 1건) 주기의 실시간 MariaDB 데이터 적재 체계 완성.
